# My Portfolio Website  

Welcome to my personal portfolio! This website showcases my web development projects, skills, and experience. Feel free to browse through my work and get in touch.

## 🚀 About This Project  
This portfolio site was built using **HTML** and **CSS** to highlight my projects, technical skills, and contact information. It demonstrates my proficiency in front-end web development, including responsive design and modern user interfaces.

### Key Features:  
- Responsive and mobile-friendly design.  
- Clean and easy-to-navigate layout.  
- A **Projects** section showcasing my work.  
- **Contact** form to get in touch with me directly.  

## 🛠️ Technologies Used  
- **HTML5**: The backbone of the website’s structure and content.  
- **CSS3**: Custom styles to bring the layout to life, including flexbox, grid, and animations.
